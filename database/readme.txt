Use the file "script.sql" in order to create the database corresponding to this current laravel project. Some doc is available in "MCD & MLD"

Utiliser le fichier "script.sql" pour créer la BD, nécessaire pour que le projet Laravel tourne. Des schémas, MCD & MLD sont disponibles dans le dossier du même nom.
En toute rigueur, il faudrait utiliser la migration de BD qui est possible grâce à Artisan et la classe mère Migration, mais nous avons préféré pour ce projet créer nos propres scripts SQL. Il suffit donc de les exécuter sur votre serveur MySQL
